# EduVid Factory — автоматическая генерация обучающих коротких видео

## Что за задача и для кого?
Для тех, кто создаёт образовательный контент в формате коротких вертикальных видео (YouTube Shorts / TikTok / VK Клипы). Сейчас производство одного ролика занимает много времени: нужно придумать сценарий, записать голос, подобрать фон, смонтировать. Особенно трудоёмко делать видео, где два персонажа объясняют сложные темы простым языком.

**EduVid Factory** решает эту проблему: это программный «завод», который автоматически генерирует такие видео. Вы задаёте тему, а скрипт сам:
- пишет диалог между двумя персонажами,
- озвучивает его голосами (синтезированными),
- накладывает фото персонажей на динамичный фон (паркур из Minecraft, Subway Surfers и т.п.),
- сохраняет готовый MP4-файл.

Проект предназначен для авторов контента, SMM-менеджеров, преподавателей, которые хотят быстро и дёшево получать виральные образовательные ролики для поколения альфа и зет.

## Что делает PoC (демо-версия)?
PoC (Proof of Concept) реализует базовый пайплайн:
- Генерация диалога на заданную тему через API OpenAI (GPT).
- Создание аудиодорожки через ElevenLabs (голоса двух разных персонажей).
- Выбор случайного фонового видео из локальной папки (Minecraft/Subway Surfers).
- Наложение двух фото персонажей на фон с помощью MoviePy.
- Сохранение готового видео в папку `output`.

PoC запускается вручную или по расписанию и делает одно видео за запуск.

## Что НЕ делает PoC (out‑of‑scope)
- Не публикует видео автоматически в соцсети (только сохраняет на диск).
- Не собирает статистику просмотров и лайков.
- Не имеет веб-интерфейса — только командная строка.
- Не поддерживает потоковую генерацию или очередь заданий.
- Не обрабатывает авторские права на голоса знаменитостей (используются обобщённые синтезированные голоса).
